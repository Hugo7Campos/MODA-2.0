# MODA 2.0 - HTML + CSS

Um projeto de site moderno com HTML5 e CSS3, focado em moda, estilo e tendências.

## 📁 Estrutura do Projeto

```
MODA 2.0/
├── index.html               # Página principal
├── sobre.html               # Página "Sobre Nós"
├── catalogos.html           # Página de catálogos
├── CSS/
│   └── style.css            # Estilos e design
└── IMAGENS/
    ├── botaa.jpg            # Imagens de produtos
    ├── bone.jpg
    ├── camisaa.jpg
    ├── calçaa.jpg
    └── blusa1.jpg
```

## 🎨 Tecnologias Utilizadas

- **HTML5** — Marcação semântica
- **CSS3** — Estilização moderna
- **Flexbox** — Layout flexível
- **Imagens** — Fotos de moda e produtos

## 🚀 Funcionalidades

- Página principal com slider de fotos
- Menu de navegação responsivo
- Seção de novidades
- Página "Sobre Nós"
- Catálogos de produtos

## 📸 Imagens

As imagens estão organizadas na pasta `IMAGENS/`:

## 📝 Como Usar

1. Abra o arquivo `index.html` no seu navegador
2. Navegue pelas páginas: Home, Sobre e Catálogos
3. Veja as tendências de moda e produtos
